===============================================================
          정리법샵 MVP 정의 (1차 출시용)
===============================================================

본 문서는 NoteGen 서비스의 정리법샵(Marketplace for Organizing Methods)
MVP 범위를 정의한다.

MVP 목표: 정리 품질 체감 향상 + 향후 크리에이터 생태계 확장 가능성 검증


===============================================================
 구현 현황 (2026-01-04 업데이트)
===============================================================

[완료]
  - 정리법 리스트 화면 (과목/정렬 필터)
  - 정리법 상세 모달 (설명, 프롬프트 표시)
  - 플랜별 정리법 잠금/해제
  - 정리법 구독/구독 해제
  - 업로드 화면에 구독한 정리법 표시
  - 좋아요(하트) 기능 + 인기순 정렬
  - 시스템 정리법 6개 (기본 요약, 코넬식, 단어장, 수학, 비교표, 시험대비)

[미구현 - 2차 MVP]
  - 사용자 정리법 생성/업로드
  - 선택형 프롬프트 조합 UI
  - 캔버스 기반 구조 에디터
  - Creator 수익 분배


===============================================================
 1. MVP 목표
===============================================================

- 사용자가 정리법을 선택해서 정리에 적용할 수 있다
- 정리법마다 필요 플랜(Free / Basic / Pro)이 명확히 표시된다
- 정리법은 구조/기능 단위로 관리되어 확장 가능하다
- 커뮤니티/수익화는 2차 MVP 이후로 미룬다


===============================================================
 2. MVP 범위 조정
===============================================================

네비게이션 구조 정의는 제외
정리법샵은 진입 경로와 무관하게 "정리법 생성/업로드/적용 경험 자체"에 초점


===============================================================
 3. 정리법샵 MVP 기능 범위
===============================================================

-----------------------------------------------------------
 3-1. 정리법 리스트 화면
-----------------------------------------------------------

필수 기능:
  - 정리법 카드 리스트
  - 정렬 옵션: 최신순 / 인기순 (하트 기반)
  - 과목 필터: 국어 / 영어 / 수학 / 사회 / 과학 / 다과목

정리법 카드 구성 요소:
  - 정리법 제목
  - 과목 태그
  - 필요 플랜 배지 (Free / Basic+ / Pro)
  - 하트 수

-----------------------------------------------------------
 3-2. 정리법 상세 화면
-----------------------------------------------------------

표시 정보:
  - 정리법 설명
  - 포함 기능 목록
  - 필요 플랜 안내 문구
  - 예상 결과물 미리보기 (스켈레톤 or 샘플)

CTA 버튼:
  - 적용하기
  - 저장하기 (북마크)

플랜 체크:
  - 내 플랜보다 상위 플랜 정리법일 경우
    -> 적용하기 비활성화
    -> 업그레이드 안내 표시


===============================================================
 4. 정리탭 연동 (핵심)
===============================================================

-----------------------------------------------------------
 4-1. 현재 정리법 표시
-----------------------------------------------------------

정리탭 상단 고정 영역:
  - 현재 정리법: [정리법 이름]
  - 변경 버튼

-----------------------------------------------------------
 4-2. 내 정리법 보기
-----------------------------------------------------------

정리탭 내에 "내 정리법" 진입 버튼 제공

내 정리법 목록 구성:
  - 내가 만든 정리법
  - 내가 저장(즐겨찾기)한 정리법

각 항목 표시 정보:
  - 썸네일
  - 정리법 이름
  - 필요 플랜 배지

-----------------------------------------------------------
 4-3. 즐겨찾기(Favorite) 동작 규칙
-----------------------------------------------------------

- 정리법샵 또는 정리법 상세에서 즐겨찾기 가능
- 즐겨찾기한 정리법은:
  - 정리탭 > 내 정리법 목록에 노출
  - 별도 검색 없이 즉시 적용 가능

즐겨찾기 = "정리할 때 바로 쓰는 정리법" 개념


===============================================================
 5. 정리법 생성 & 업로드 (MVP 핵심)
===============================================================

-----------------------------------------------------------
 5-1. 정리법 구성 요소
-----------------------------------------------------------

정리법 = Prompt + Structure + Features

  1) Prompt (프롬프트)
  2) Structure (구조)
  3) Features (기능)

MVP부터 사용자 직접 정리법 생성 및 업로드 허용

-----------------------------------------------------------
 5-2. 썸네일 설정
-----------------------------------------------------------

- 사용자는 정리법 업로드 시 썸네일 이미지 추가 가능
- 이미지 미등록 시: 기본 제공 이미지 세트 중 랜덤 자동 선택

권장 규격:
  - 비율: 1:1
  - 텍스트 최소화 (카드 가독성 우선)

-----------------------------------------------------------
 5-3. 프롬프트 작성 방식 (혼합형)
-----------------------------------------------------------

사용자는 두 가지 방식 중 선택 가능:

[A] 직접 작성 (고급 사용자)
    - 자유 텍스트 입력
    - 전체 프롬프트 커스터마이징 가능

[B] 선택형 조합 (기본 사용자)
    - 체크/드롭다운으로 프롬프트 구성

    선택 항목 예시:
      [ ] 단어 뜻 정리
      [ ] 다양한 의미
      [ ] 비슷한 단어 / 유의어
      [ ] 반의어
      [ ] 예문 포함 여부
      [ ] 시험 대비 여부

    선택형 조합은 내부적으로 프롬프트 템플릿으로 변환됨

-----------------------------------------------------------
 5-4. 구조 지정 (단계적 확장)
-----------------------------------------------------------

[MVP 단계]
  - 미리 정의된 구조 템플릿 선택
    - 요약형
    - 단어장형
    - 비교표 중심

[차기 단계 - 확장 아이디어]
  - 사용자 지정 캔버스 제공
  - 영역 단위 지정
    - 키워드 영역
    - 요약 영역
    - 문제 영역

  -> 추후 캔버스 기반 정리법 에디터로 확장 가능


===============================================================
 6. 기능별 플랜 판정 로직
===============================================================

-----------------------------------------------------------
 6-1. 기능 메타데이터
-----------------------------------------------------------

각 기능은 최소 플랜을 가진다:

  [Free 기능]
    - 개념 요약
    - 핵심 강조

  [Basic 기능]
    - 단어 추출
    - 예문 생성
    - 비교표

  [Pro 기능]
    - 시험형 문제
    - 출제자 관점

-----------------------------------------------------------
 6-2. 정리법 플랜 계산 규칙
-----------------------------------------------------------

정리법에 포함된 기능 중 가장 높은 플랜 = 정리법 필요 플랜

예시:
  - 단어 + 뜻 + 예문 -> Basic
  - 예문 + 시험 문제 -> Pro


===============================================================
 7. 예상 결과물 미리보기 (MVP)
===============================================================

미리보기 방식:
  - 구조 기반 스켈레톤 미리보기
  - 샘플 텍스트 기반 간단 생성 (저비용)

실제 노트 테스트는 MVP 제외


===============================================================
 8. MVP 제외 항목 (명확히 제외)
===============================================================

  - 네비게이션 구조 고정
  - 월간/올타임 랭킹
  - Creator 수익 분배
  - 고급 캔버스 에디터 (영역 직접 그리기)


===============================================================
 9. MVP 성공 판단 기준
===============================================================

  - 정리 실행 시 정리법 선택 비율
  - 정리법 변경 빈도
  - Basic -> Pro 업그레이드 전환율
