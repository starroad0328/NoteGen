    ✓ 새로운 공부법을 만들지 않는다
    ✓ 이미 검증된 방법을 AI로 자동화한다
    ✓ 사용자가 선택한 정리법을 정확히 따른다

  ⚠️  핵심: AI가 멋대로 바꾼 게 아니라
     "내가 선택한 방식"으로 정리됐다는 느낌

[ 사용자는 이렇게 느껴야 함 ]

  필기 업로드    →  사진/PDF/텍스트
       ↓
  정리법 선택    →  코넬식? 아웃라인? 마인드맵? Q&A?
       ↓
  옵션 설정      →  학년, 손글씨 스타일 강도
       ↓
  정리하기 클릭  →  AI 자동 정리
       ↓
  결과 확인      →  손글씨 느낌의 깔끔한 노트


  🎯 목표: "내가 선택한 방식대로 잘 정리됐네!"

[ AI 처리 로직 ]

  ✏️  본문 영역
     → 필기 내용을 문단별로 재정리
     → 명확한 문장 구조로 변환

  🔑 키워드 영역
     → 핵심 개념 추출
     → 질문형으로 변환 ("~란?", "~의 특징은?")

  📝 요약 영역
     → 전체 내용을 3~5줄로 압축
     → 시험 직전 복습용

[ AI 처리 로직 ]

  ❓ 질문 생성

     → 핵심 키워드 강조

  🎯 시험 대비 최적화
     → 암기할 내용 중심
     → 헷갈리기 쉬운 부분 구분

다음 기능들은 V2/V3에서 구현:

  ❌ 유저 필기체 개인화 학습
     → MVP는 범용 손글씨 폰트 사용

  ❌ 그래픽 마인드맵 생성
     → MVP는 텍스트 기반만

  ❌ 문제 자동 채점
     → 정리 기능만 제공

  ❌ 학습 진도 분석
     → 트래킹 기능 없음

  ❌ 과목별 자동 추천
     → 사용자가 직접 선택

  ❌ AI 튜터 대화
     → 정리 결과 제공만


  ⚠️  이유: MVP는 "정리 자동화"에만 집중

┌─────────────────────────────────────────────────────────┐
│  예시: 코넬식 정리법 결과                               │
└─────────────────────────────────────────────────────────┘

┌────────────────┬────────────────────────────────────┐
│                │  📌 일차함수                       │
│ 일차함수란?    │                                    │
│                │  일차함수는 y = ax + b 형태의      │
│ 기울기 a?      │  함수입니다.                       │
│                │                                    │
│ y절편 b?       │  • a: 기울기 (x 변화 대비 y 변화) │
│                │  • b: y절편 (x=0일 때 y값)        │
│ 예시?          │                                    │
│                │  ✏ 예시                            │
│                │  y = 2x + 3                        │
│                │  → x가 1 증가하면 y는 2 증가       │
│                │  → x=0일 때 y=3                    │
├────────────────┴────────────────────────────────────┤
│  📝 요약                                             │
│  일차함수 y=ax+b에서 a는 기울기, b는 y절편.         │
│  기울기가 클수록 그래프가 가파르다.                  │
└──────────────────────────────────────────────────────┘


┌─────────────────────────────────────────────────────────┐
│  예시: 아웃라인 정리법 결과                             │
└─────────────────────────────────────────────────────────┘

1. 일차함수
   1.1 정의
       y = ax + b 형태의 함수

   1.2 구성 요소
       1.2.1 기울기 (a)
             • x가 변할 때 y의 변화율
             • 그래프의 기울기 결정

       1.2.2 y절편 (b)
             • x=0일 때 y의 값
             • 그래프가 y축과 만나는 점

   1.3 예시
       y = 2x + 3
       • a = 2 (기울기)
       • b = 3 (y절편)


┌─────────────────────────────────────────────────────────┐
│  예시: Q&A 정리법 결과                                  │
└─────────────────────────────────────────────────────────┘

Q1. 일차함수란 무엇인가?
A.  y = ax + b 형태의 함수로, x의 일차식으로
    표현되는 함수입니다.

Q2. 기울기 a는 무엇을 의미하는가?
A.  x가 1 증가할 때 y의 변화량을 의미합니다.
    그래프의 기울기를 결정합니다.

Q3. y절편 b는 무엇인가?
A.  x=0일 때 y의 값입니다.
    그래프가 y축과 만나는 점의 y좌표입니다.

Q4. y = 2x + 3 그래프의 특징은?
A.  • 기울기: 2 (x가 1 증가 시 y는 2 증가)
    • y절편: 3 (0, 3)을 지남
