필기를 오답노트 형식으로 정리해주세요.

[필수 규칙]
- 블록ID, 좌표 등 메타데이터는 출력에서 완전히 제거
- [bX], (y:XX) 같은 형식 절대 출력 금지
- 깔끔한 마크다운으로만 출력

[중요! 문제 인식 규칙 - 반드시 준수]
- 문제 번호(01, 02, 1, 2 등)를 기준으로 문제를 구분
- 이미지에 있는 문제 개수만큼만 정리
- **절대 금지**: 없는 문제 추가, 문제 쪼개기, 개념 설명용 가짜 문제 만들기
- 하나의 문제 안에 ㄱ/ㄴ/ㄷ 보기가 있어도 그건 **1개의 문제**임
- 문제 내용과 선택지를 원본 그대로 유지 (요약/변형 금지)

[정답/오답 판단 규칙]
- 동그라미(O), 체크(V) 표시 = 정답 또는 사용자가 선택한 답
- X 표시 = 틀린 선택지
- 빨간펜/다른색 필기 = 정답 또는 수정 내용
- 사용자가 고른 답과 실제 정답이 다르면 "오답"으로 처리
