주제: 코넬식 노트 UI 개선 (줌 캔버스)

 작업 내용
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 코넬식 노트 레이아웃 전면 개편
   - 기존: 단순 마크다운 렌더링
   - 변경: 큰 종이(도화지) 형태의 캔버스
   - 핀치 줌으로 확대/축소 가능

2. 줌 기능 구현
   - @openspacelabs/react-native-zoomable-view 라이브러리 사용
   - 캔버스 크기: 화면의 2배 너비, 2.5배 높이
   - 줌 범위: 0.4x ~ 2.5x
   - 경계 제한 (bindToBorders)으로 캔버스 밖으로 이동 방지

3. 코넬 노트 형식 파싱
   - 새 형식: ===TITLE===, ===KEYWORDS===, ===NOTES===, ===SUMMARY=== 마커
   - 기존 형식: 헤딩에서 키워드 추출, **요약** 섹션 추출
   - 노트 영역에 전체 정리 내용 표시

4. 종이 스타일 UI
   - 줄 노트 라인 (가로선)
   - 왼쪽 빨간 여백선 (키워드 영역)
   - 하단 요약 영역
   - 상단 제목 영역

코넬식 프롬프트 출력 형식:
  ===TITLE===
  제목

  ===KEYWORDS===
  - 키워드 목록

  ===NOTES===
  본문 내용

  ===SUMMARY===
  요약

줌 설정:
  - initialZoom: 0.5 (전체 보기)
  - maxZoom: 2.5
  - minZoom: 0.4
  - bindToBorders: true (경계 제한)
  - panBoundaryPadding: 50 (여유 공간)
