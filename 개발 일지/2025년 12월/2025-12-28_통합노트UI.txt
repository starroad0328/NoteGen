 2. 블록 타입 시스템
-----------------------------------------------------------

기본 블록 (12개):
  - title: 제목
  - heading: 소제목 (level 1/2/3)
  - paragraph: 본문
  - bullet: 글머리표
  - numbered: 번호 목록
  - keyword: 핵심 키워드 (코넬식 cues)
  - summary: 요약
  - important: 중요 강조
  - example: 예시
  - formula: 공식
  - definition: 정의
  - tip: 팁/주의
  - divider: 구분선
  - simpleSection: 단순 섹션 (라벨 + 내용 + 구분선)

오답노트 블록 (4개):
  - problem: 문제
  - solution: 정답/풀이
  - wrongPoint: 틀린 포인트 (내 답 + 이유 + 올바른 접근)
  - concept: 관련 개념

단어장 블록 (1개):
  - vocabulary: 단어 (word, meaning, pronunciation, etc.)


-----------------------------------------------------------
 3. 오답노트 마크다운 패턴 인식
-----------------------------------------------------------

converter.ts에서 자동 감지하는 패턴:

  **문제**:        -> ProblemBlock
  **내 풀이**:     -> WrongPointBlock.myAnswer
  **오답**:        -> tip(warning)
  **정답**:        -> SolutionBlock
  **틀린 이유**:   -> WrongPointBlock.reason
  **핵심 공식**:   -> SimpleSectionBlock
  **주의점**:      -> SimpleSectionBlock
  **관련 개념**:   -> ConceptBlock

인식 조건:
  - metadata.organizeMethod === 'wrong_answer'
  - 또는 마크다운에 위 패턴이 포함됨
