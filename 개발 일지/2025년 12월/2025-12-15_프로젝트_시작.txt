  │   │   └── notes.py         ✅ 노트 관리 API
  │   ├── core/
  │   │   ├── config.py        ✅ 환경 설정
  │   │   ├── database.py      ✅ DB 연결
  │   │   └── pricing.py       ✅ 가격 정책
  │   ├── models/
  │   │   ├── note.py          ✅ Note 모델
  │   │   └── user.py          ✅ User 모델
  │   ├── schemas/
  │   │   └── note.py          ✅ Pydantic 스키마
  │   └── services/
  │       ├── ocr_service.py   ✅ OCR 서비스
  │       └── ai_service.py    ✅ AI 정리 서비스


[ API 엔드포인트 ]

  POST   /api/upload/              이미지 업로드
  GET    /api/upload/{id}          업로드 정보 조회
  POST   /api/process/{id}/process 처리 시작
  GET    /api/process/{id}/status  처리 상태 확인
  GET    /api/notes/               노트 목록
  GET    /api/notes/{id}           노트 상세
  DELETE /api/notes/{id}           노트 삭제


[ 파이프라인 플로우 ]

  1. 이미지 업로드 → DB 저장
  2. OCR 처리 → 텍스트 추출
  3. AI 정리 → 마크다운 생성
  4. 결과 저장 → 완료
