Backend 실행:
  cd backend
  pip install -r requirements.txt
  uvicorn app.main:app --reload
  → http://localhost:8000/docs

Mobile App 실행:
  cd mobile
  npm install
  npx expo start
  → Expo Go 앱으로 QR 스캔
