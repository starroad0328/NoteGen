필수:
  □ OpenAI API Key 발급
    → https://platform.openai.com/api-keys

  □ backend/.env 파일에 API 키 입력
    → OPENAI_API_KEY=발급받은키

선택:
  □ Google Cloud Vision API (더 나은 OCR)
  □ Android Studio (에뮬레이터 테스트)

 빠른 실행 명령어

Backend 실행:
  cd backend
  pip install -r requirements.txt
  uvicorn app.main:app --reload
  → http://localhost:8000/docs

Mobile App 실행:
  cd mobile
  npm install
  npx expo start
  → Expo Go 앱으로 QR 스캔
