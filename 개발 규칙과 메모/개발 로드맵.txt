     - 오버레이 탭으로 닫기

  4. 백엔드 API 업데이트
     - NoteResponse에 image_urls 필드 추가
     - get_image_urls() 헬퍼 함수 구현
     - 정적 파일 서빙 (/uploads)

  5. 탭 네비게이션 구조 정리
     - (tabs) 폴더로 통합
     - 기존 중복 파일 삭제 (index.tsx, upload.tsx 등)

  - 탭 기반 UI (핵심/설명/시험/원본)
